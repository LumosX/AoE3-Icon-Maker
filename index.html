Alpha Mask

Greyscale mask to use for transparency. White = opaque, black = transparent. If you upload a transparent PNG, it will be automatically converted to a mask.

Unit name

Name to use in the filenames of exported images. If left blank, will be automatically derived from an uploaded file.

Quick export

Download the game-ready portrait with alpha mask, or just the portrait RGB (without transparency) or mask. Since Photoshop "some image editors" don't readily support PNG alpha channels, the latter two options are useful if you want to split a portrait image extracted from the game.

Standard frames

You may choose individual icons to download. Hover over each for usage and examples. Icon size is 128x128 px.

Mixed frames

Pick a primary (top-right) colour and a secondary (bottom-left) colour to generate mixed frames. A colour will not combine with itself. All mixed-frame icons you see will be considered "selected" for bulk download.
